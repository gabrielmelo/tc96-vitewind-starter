## Installation:

**Prerequisite**
- Nodejs
- npm or pnpm

With npm
```sh
npm install
```

With pnpm
```sh
pnpm i
```

Quick commands

```sh
# dev server start
npm run dev or pnpm dev
```

```sh
# production build
npm run build or pnpm build
```
